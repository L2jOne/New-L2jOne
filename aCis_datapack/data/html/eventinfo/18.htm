Treasure Hunt
Event Information
A few treasure chests spawn in random locations at elven ruins.
The goal is to be the first to find a treasure chest.
A player can find more than 1 chest.
For each chest opened player gets reward.
The first to find a treasure gets extra reward.
There is a small chance of getting a random S-grade key material when opening a chest.
